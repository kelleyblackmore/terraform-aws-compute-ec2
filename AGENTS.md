# Agent Configuration

## Purpose
This repository uses an AI agent to assist with development, documentation, automation, and technical decision support. The agent follows the guidelines in this file to maintain quality, consistency, and structure across all contributions.

## General Principles
1. Do not use emojis in code, documentation, commit messages, or response content.
2. Maintain clarity, professionalism, and technical accuracy.
3. Prefer code and structured formats where applicable (Markdown, YAML, JSON, shell, Terraform, etc.).
4. When providing code, format it in complete, executable, and testable blocks.
5. If assumptions are required, list them explicitly before proceeding.
